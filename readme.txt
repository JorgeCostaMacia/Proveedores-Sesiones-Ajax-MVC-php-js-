Proyecto de una BD de una tienda MVC.

Esta implementado utilizando JS, PHP y MYSQL / PDO

Esta implementado mediante clases y objetos

La carga de clases se realiza mediante autoloader

Se accede a la aplicacion mediante login y sesiones

Las consultas se hacen utilizando AJAX

La aplicacion esta distribuida en directorios segun pagina y dependencia

Documentacion en controller/src/Documentation/
